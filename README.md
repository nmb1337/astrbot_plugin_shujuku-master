# 剧本杀 NPC 数据库

一个 AstrBot 群聊剧本杀辅助插件，提供 NPC 数据库、星币流通、每日打卡、角色状态栏、月度奖池抽奖和九宫格物品栏。主要回执都会生成图片展示。

## 功能

- 星币：群员可查询余额，也可赠送星币给其他群员。
- 打卡：每天一次，随机获得 1-3 星币或 1 张免费入场券。
- 状态栏：展示当前 NPC 的星级、经验进度、立绘、技能、名称和介绍。
- 切换角色：把已拥有 NPC 切换为当前状态栏角色。
- 抽奖：10 星币一次；单抽时免费入场券会优先抵扣。大奖池每月自动轮换。
- 物品栏：九宫格展示当前拥有和未拥有的 NPC。
- NPC 信息：查询 NPC 获取途径、经验加成、介绍和技能。

## 指令

```text
/剧本杀帮助
/星币
/赠送星币 @群友 20
/打卡
/状态栏
/切换角色 星见凛
/抽奖
/抽奖 10
/物品栏
/NPC信息 星见凛
```

## 打卡概率

- 1 星币：45%
- 2 星币：30%
- 3 星币：20%
- 免费入场券 1 张：5%

## 抽奖规则

- 每抽消耗 10 星币。
- 单抽时，如果有免费入场券，会优先消耗 1 张入场券。
- 大奖概率为 6%，从本月大奖池中抽取 NPC。
- 抽到未拥有的大奖 NPC 会解锁该 NPC。
- 抽到已拥有的大奖 NPC 会转化为 600 点该 NPC 经验。
- 其余奖品为礼物，会自动转化为当前角色经验。

## 经验等级

总共 5 星，每级所需经验：

```text
1星进度：1000
2星进度：2000
3星进度：3000
4星进度：4000
5星进度：5000
满进度累计：15000
```

状态栏会显示当前星级、当前等级进度条，以及 2 星、3 星、5 星技能。未解锁技能会灰色显示。

## 数据与图片

插件会在本目录自动创建：

```text
data/players.json          玩家数据
data/npc_assets/*.png      NPC 立绘
data/rendered/*.png         回复图片缓存
```

默认立绘是插件首次启动时用 Pillow 生成的二次元风格占位图。你可以把同名 PNG 替换为自己的角色图，例如：

```text
data/npc_assets/rin.png
data/npc_assets/yue.png
data/npc_assets/mika.png
```

替换后再次使用 `/状态栏`、`/物品栏`、`/NPC信息` 就会使用新立绘。

## 依赖

需要 Pillow：

```bash
pip install pillow
```

如果 AstrBot 运行环境已经安装 Pillow，则不需要额外操作。
