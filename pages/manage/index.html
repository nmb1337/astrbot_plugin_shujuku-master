剧本杀同伴运营面板

按顺序维护总库、角色模板和玩家资产。所有图片保存在插件本地，无需图床。

第一板块

角色、皮肤、道具与经验球

经验球属于抽奖奖励，直接增加当前同伴经验，不会进入道具栏。

色彩控制

先选择要调整的图片类型，只显示对应色彩，避免混杂。

总库

编辑条目

条目图片上传选择、拖放，或点击后 Ctrl+V 粘贴图片。
未选择条目

上传图片后可调整画面焦点。

第二板块

角色绑定的打卡背景与文字模板

优先匹配当前皮肤，其次当前同伴，最后使用通用模板。每个模板只保留一张全背景，画框和排版由背景图完成。

可用变量：{title}、{reward}、{coins}、{message}、{date}、{user_name}、{user_id}。{message} 的位置由下方文字行控制。
字段 / 文本（可用变量） / X / Y / 字号 / 字体 / 颜色 / 加粗
上传全背景16:9 图片会无缝覆盖整张打卡卡片。
第三板块

角色绑定的状态栏模板

模板只保留全背景与文字。经验条已移除；同伴或皮肤填写的经验加成、2/3/5 星技能可在文字行中绑定并调位置。

可用变量：{user_name}、{user_id}、{character_name}、{subtitle_name}、{quality}、{bonus}、{stars}、{level}、{current}、{need}、{skill_2_name}、{skill_2_desc}、{skill_3_name}、{skill_3_desc}、{skill_5_name}、{skill_5_desc}。带“绑定”字样的默认行会读取当前同伴；装备皮肤后优先读取皮肤填写的内容。
字段 / 文本（可用变量） / X / Y / 字号 / 字体 / 颜色 / 加粗
上传全背景图片会覆盖整个状态栏。
第四板块

玩家资产发放与扣除

可发放或收回同伴、皮肤和道具；道具可按数量扣除。经验球不会出现在此处。
