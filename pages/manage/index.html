剧本杀同伴运营面板

按顺序维护总库、角色模板和玩家资产。所有图片保存在插件本地，无需图床。

第一板块

角色、皮肤、道具与经验球

经验球属于抽奖奖励，直接增加当前同伴经验，不会进入道具栏。

色彩控制

先选择要调整的图片类型，只显示对应色彩，避免混杂。

总库

编辑条目

条目图片上传选择、拖放，或点击后 Ctrl+V 粘贴图片。
未选择条目

上传图片后可调整画面焦点。

第二板块

角色绑定的打卡背景与文字模板

优先匹配当前皮肤，其次当前同伴，最后使用通用模板。背景、文字面板与角色框可独立上传。

可用变量：{title}、{reward}、{coins}、{message}、{date}、{user_name}、{user_id}。全尺寸透明图会完整叠加；较小图片会自动放到对应区域。
字段 / 文本 / X / Y / 字号 / 字体 / 字重 / 颜色 / 阴影 / 删除
上传全背景16:9 图片会无缝覆盖整张打卡卡片。
上传文字面板层可使用透明 PNG 装饰文字区域。
上传角色框层可使用透明 PNG 边框。
第三板块

角色绑定的状态栏模板

保持原有左角色、右信息的状态栏排版；三层图片、角色绑定、字体和打开者信息都可单独配置。

可用变量：{user_name}、{user_id}、{character_name}、{subtitle_name}、{quality}、{bonus}、{stars}、{level}、{current}、{need}。
字段 / 文本 / X / Y / 字号 / 字体 / 字重 / 颜色 / 阴影 / 删除
上传全背景图片会覆盖整个状态栏。
上传信息面板层可使用透明 PNG 装饰右侧信息区。
上传角色框层可使用透明 PNG 装饰左侧角色区。
第四板块

玩家资产发放与扣除

可发放或收回同伴、皮肤和道具；道具可按数量扣除。经验球不会出现在此处。
